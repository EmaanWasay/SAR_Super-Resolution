# SAR Super Resolution

This project implements deep learning models (SRCNN and Residual Networks) for SAR image super-resolution.

## AI Usage
- AI (ChatGPT model 5.3 and 5.4 have been used to research on the topic to search and study similar research articles and to find the proper model for this project and for programming section.
- Help from Claude is taken in order to write IEEE format report and adding references moreover overleaf is used and Latex was used for IEEE report preparation.
